Enterprises

Project that manage the hierarchy of three main entities: Enterprises -> Departments -> Employees. The main goal is to create database model with all main relationships(manyToOne, manyTomany etc.) and test on it different data access DAO realisations(JDBC, Hibernate, SpringJDBC etc.). WEB part is implemented with SpringMVC and JSP. So the project has MVC structure with 3 layers: DAO, Services, Controllers.
